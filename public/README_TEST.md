# E-Commerce Platform

A full-stack e-commerce solution built with the MERN stack.

## Features

- User authentication with JWT
- Shopping cart functionality
- Stripe payment integration
- Order management
- Admin dashboard
- Responsive design

## Tech Stack

- React
- Node.js
- Express
- MongoDB

## Getting Started

```bash
# Install dependencies
npm install

# Run the app
npm run dev
```
